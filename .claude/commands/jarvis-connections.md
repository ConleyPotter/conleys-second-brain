# JARVIS — Weekly Connections

Read `Conley's 2nd Brain/05-CLAUDE/CLAUDE.md` in full before doing anything else.
Then read and execute the skill at `Conley's 2nd Brain/05-CLAUDE/skills/weekly-connections.md`.

Surface 3–5 strong connections from notes added to `Conley's 2nd Brain/01-CAPTURES/` in the last 7 days.
Create connection notes in `Conley's 2nd Brain/02-CONNECTIONS/`.

Only surface connections that would genuinely surprise the person who wrote the notes. If a connection is obvious, it does not qualify.
