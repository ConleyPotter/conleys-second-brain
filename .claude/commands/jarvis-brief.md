# JARVIS — Generate Brief

Read `Conley's 2nd Brain/05-CLAUDE/CLAUDE.md` in full before doing anything else.
Then read and execute the skill at `Conley's 2nd Brain/05-CLAUDE/skills/generate-brief.md`.

Generate a content brief for the connection or topic Conley specifies.

If no topic is specified, ask which connection note from `Conley's 2nd Brain/02-CONNECTIONS/` to brief.

Save the brief to `Conley's 2nd Brain/03-BRIEFS/` with today's date and topic as filename.
Example: `2026-04-25-pipeline-failure-at-2am.md`
Tag it `#ready-to-write`.
