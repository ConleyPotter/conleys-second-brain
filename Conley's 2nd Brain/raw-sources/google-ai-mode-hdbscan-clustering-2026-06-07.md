# HDBSCAN clustering — Google AI Mode

HDBSCAN (Hierarchical Density-Based Spatial Clustering of Applications with Noise) is an unsupervised machine learning algorithm that groups data points by identifying dense regions. It extends standard DBSCAN by finding clusters of varying densities and automatically determining the optimal number of clusters while effectively handling noise and outliers.

## How HDBSCAN Works

Instead of requiring you to guess a global density threshold (like ε in traditional DBSCAN), HDBSCAN builds a hierarchy of clusters.

1. **Mutual Reachability Distance:** It transforms the space between points using a metric called mutual reachability, which penalizes points in low-density areas.
2. **Minimum Spanning Tree:** It connects all data points into a tree based on their mutual reachability distances.
3. **Cluster Hierarchy:** It breaks the tree apart to build a hierarchy of potential clusters.
4. **Stable Cluster Extraction:** It condenses the hierarchy tree by keeping only the clusters that are "most persistent" and stable over a wide range of density scales, discarding transient groupings.

## Key Advantages

- **Variable Density:** Unlike DBSCAN, it can identify clusters with drastically different densities within the same dataset.
- **Noise Tolerance:** Points that do not fall into any dense, cohesive group are automatically classified as noise (often labeled as -1).
- **Intuitive Tuning:** It generally requires minimal parameter tuning. The most important parameter is `min_cluster_size`. Increasing this value merges smaller, noisy clusters into larger, more stable ones or into noise.
- **Soft Clustering:** It provides probabilities or membership vectors, indicating how confident the algorithm is that a point belongs to a specific cluster (rather than a hard assignment).

## Basic Implementation (Python)

Implementing HDBSCAN follows the standard scikit-learn API design. You can use the popular `hdbscan` package provided by the scikit-learn-contrib community.

Install the library:

```bash
pip install hdbscan
```

Run a simple clustering script on synthetic data:

```python
import hdbscan
from sklearn.datasets import make_blobs
import matplotlib.pyplot as plt

# 1. Generate sample data
data, _ = make_blobs(n_samples=500, centers=5, cluster_std=0.5, random_state=42)

# 2. Initialize the HDBSCAN clusterer
clusterer = hdbscan.HDBSCAN(min_cluster_size=15, min_samples=5)

# 3. Fit the data and extract labels
cluster_labels = clusterer.fit_predict(data)

# 4. Plot the results
plt.scatter(data[:, 0], data[:, 1], c=cluster_labels, cmap='Spectral', s=50)
plt.title("HDBSCAN Clustering")
plt.show()
```

## Common Use Cases

- **Topic Modeling:** Grouping text embeddings (e.g., BERTopic relies heavily on HDBSCAN).
- **Geospatial Analytics:** Identifying dense population centers, delivery zones, or movement patterns.
- **Genomics & Bioinformatics:** Finding clusters of genetic expression without knowing the exact number of groupings beforehand.
- **Dimensionality Reduction Analysis:** Clustering embedded data points generated by algorithms like UMAP.
